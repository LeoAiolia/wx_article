# 推荐一个超硬核的 AI 编程 CLI 终极中文指南（Claude Code + Codex）

> 导语：从“网页聊天”到“终端 Agent”，AI 辅助编程正在迎来范式转移。今天给大家分享一个超硬核的开源项目，带你彻底打通 Claude Code 与 Codex 的终端开发密码。

---

## 前言

作为移动端（iOS / Flutter）开发者，你平时是怎么用 AI 写代码的？

是在浏览器里打开 ChatGPT/Claude 网页端复制粘贴代码？还是依赖 IDE 插件（如 Copilot、Cursor）进行单文件级的补全？

如果你还在用这些方式，那么你可能错过了 AI 编程的最前沿变革——**基于终端（CLI）的自主智能体（Coding Agent）**，比如最近大火的 **Claude Code** 和 OpenAI 最新推出的 **Codex CLI**。

最近，国内著名技术博主 stormzhang（张涛）开源了一份重磅教程：[ai-coding-guide](https://github.com/stormzhang/ai-coding-guide)，在社区获得了极高的关注度。今天不做长篇大论，直接把这个宝藏教程分享给大家。

* 官网在线阅读地址：[coding.stormzhang.ai](https://coding.stormzhang.ai/)
* GitHub 仓库地址：[github.com/stormzhang/ai-coding-guide](https://github.com/stormzhang/ai-coding-guide)

---

## 一、这份指南有什么干货？

这份教程包含 **92 篇精修文章，约 52 万字**，系统性地涵盖了当前主流的两个终端 AI 编程工具：
* **Claude Code 篇（53 篇）**：包含基础安装、API 配置，以及子代理（Subagents）、模型上下文协议（MCP）、Agent Skills、自动触发的 Hooks 等高级玩法。
* **Codex 篇（39 篇）**：深入剖析 OpenAI 新版 Codex CLI 的工作原理、沙箱配置、读写权限控制以及日常 Git 工作流的整合。

---

## 二、为什么推荐你阅读？

* **以官方文档为准**：避开网上零散的猜测，所有功能命令均对照官方文档核实。
* **面向小白改写**：采用“场景引入 + 生活类比 + 实际场景”的教学，每篇 3~10 分钟，并提供可照跑的完整命令和预期输出。
* **极致终端美学**：在线阅读网站专门设计了极具极客氛围的暗色终端风界面，配有 81 张原创 SVG/PNG 插图，非常丝滑。

对于多平台混合、构建流程繁琐的 Flutter 或 iOS 开发者来说，熟练掌握这类终端 Agent 并在项目中配置规则（如 `CLAUDE.md`）和触发 Hook，能让开发效率获得降维打击。
